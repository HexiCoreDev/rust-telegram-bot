## Acknowledgments

This project is a faithful Rust port of [python-telegram-bot](https://github.com/python-telegram-bot/python-telegram-bot). The architecture, handler system, filter composition, conversation state machine, job queue design, and persistence layer are all direct translations of the Python original.

We thank the python-telegram-bot maintainers and community -- especially Leandro Toledo de Souza and all contributors -- for over a decade of work that made this project possible.
